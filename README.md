# Cyberpunk2126 WebTimer

架空アニメーション「Cyberpunk2126」の世界観を表現した、プロモーション用Webタイマーです。
**視覚（ネオン演出）＋音（BGM/効果音）＋演出（終了時メッセージ）**を統合して制作しました。

## Demo
- GitHub Pages：git@github.com:202510-aw-oi/Cyberpunk2126-WebTimer.git

## 概要 / コンセプト
- 作品名：Cyberpunk2126 プロモーション用Webタイマー
- コンセプト：宣伝用途を想定し、サイバーパンク世界観を「視覚＋音＋演出」で体験できるタイマーとして企画・制作

## 主な機能
- ネオン虹色アニメーション（CSS keyframes）
- 分・秒の動的設定（入力パネル）
- BGM / 効果音のON・OFF制御
- カウント終了後の演出切替（メッセージ＋サウンドシーケンス）
- フェード演出（メッセージ表示）

## 技術的工夫点
- 音声の `ended` イベントを利用したサウンドシーケンス制御
- 固定値 / 可変値を分離したタイマー設計
- UIとロジックを整理し、可読性を意識した構成
- GitHubでのバージョン管理（SourceTree使用）

## 使用技術
- HTML / CSS / JavaScript
- VSCode / SourceTree / GitHub
- 生成AI（プロンプト設計：キービジュアル作成）
- フリー音源活用

## ローカル起動方法
1. リポジトリをクローン
2. `index.html` をブラウザで開く
